# homebrew-harborrs

Homebrew tap for [harborrs](https://github.com/kfet/harborrs) — a self-hosted
single-binary RSS server speaking the Google Reader API dialect.

## Install (macOS & Linuxbrew)

```sh
brew tap kfet/harborrs
brew install kfet/harborrs/harborrs
```

## Upgrade

```sh
brew upgrade harborrs
```

## Not using Homebrew?

For curl-pipe install, `go install`, source builds, release tarballs, and
everything else, see the main repo:
**[github.com/kfet/harborrs](https://github.com/kfet/harborrs)**.

## How this tap is maintained

`Formula/harborrs.rb` is auto-generated by the release workflow in
[`kfet/harborrs`](https://github.com/kfet/harborrs) on every tagged release,
from `packaging/homebrew/harborrs.rb.tmpl` with SHA256s of the release
tarballs substituted in. Don't edit it here by hand — your changes will be
overwritten on the next release.
